## Installation

To install and apply the theme:


    First, create the necessary directory structure:
    ```bash
    mkdir -p ~/.config/opencode/themes
    ```
    Next, create or open the `mountain.json` file using your preferred text editor (e.g., Vim) and paste the theme's JSON content into it:
    ```bash
    vim ~/.config/opencode/themes/mountain.json
    ```

2.  **Apply the theme in OpenCode:**

    *   Launch OpenCode.
    *   run `/themes` 
    *   Select `mountain` from the list of available themes to activate it.
